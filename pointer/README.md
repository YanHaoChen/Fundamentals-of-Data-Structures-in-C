## 注意事項：



* &：取址運算子

* \* ：取值運算子（提取指向的記憶體的值）
* 指標變數大小固定：同定址能力
* 指標不能直接給定初值
* 指標變數間不能相加
* 相同型態的指標變數間可以相減：可得兩者間可使用的單位個數
