# socket

學習使用，在 TCP 及 UDP 協定下的 socket。

## TCP

檔案：[tcp](https://github.com/YanHaoChen/Learning-C/tree/master/socket/tcp)

### 執行方式

```shell
# 編譯
$ gcc server.c -o server.o
$ gcc client.c -o client.o

# 執行
$ ./server.o
$ ./client.o
```
